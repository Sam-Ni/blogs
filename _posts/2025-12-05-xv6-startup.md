# Installation of xv6-riscv

This article is about installation fo xv6-riscv.

## Get xv6-riscv source code

Get to the [xv6-riscv](https://github.com/mit-pdos/xv6-riscv) repository and clone the code to your machine.

```
git clone git@github.com:mit-pdos/xv6-riscv.git
```

## Install QEMU and C Compiler for riscv

Please follow the steps in the [MIT](https://pdos.csail.mit.edu/6.1810/2024/tools.html)
to install RISC-V versions of QEMU 7.2+, GDB 8.3+, GCC, and Binutils
according to your OS (Linux, macOS, or Windows(wsl)).
